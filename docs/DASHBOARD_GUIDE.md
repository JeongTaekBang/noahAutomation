# NOAH Dashboard 사용 가이드

---

## 1. 소개

NOAH 인터컴퍼니 거래 현황을 한눈에 파악할 수 있는 대시보드입니다.
Excel로 관리 중인 수주(SO)/발주(PO)/출고(DN) 데이터를 자동으로 시각화합니다.

### 주요 기능 (8페이지)

| 페이지 | 설명 |
|--------|------|
| 오늘의 현황 | 납기 캘린더, EXW 출고 지연, 선적 Action Items, 세금계산서 미발행 |
| 수주/출고 현황 | 월별 추이, Book-to-Bill, 일별 출고 |
| 제품 분석 | 제품별 매출 Top 15, 구성비, 평균단가, Backlog |
| 섹터 분석 | 섹터별 매출, 제품 믹스, 평균 주문 규모 |
| 고객 분석 | 고객별 매출, Pareto, 월별 추이 |
| 발주 커버리지 | 미발주/부분발주/발주진행중/발주확정, PO Status 파이프라인 |
| 수익성 분석 | 매출 vs 원가(ICO) 마진, 저마진 경보, 미출고금액 |
| Order Book | 워터폴 차트, Aging 분석, 리드타임 분석, 전환 퍼널 |

---

## 2. 사용 방법

**별도 프로그램 설치가 필요 없습니다.**

### 실행

1. 공유받은 **NOAH_Dashboard** 폴더를 본인 PC에 복사합니다.
2. **NOAH Dashboard.bat** 파일을 더블클릭합니다.
3. 잠시 후 브라우저에 대시보드가 자동으로 열립니다.
4. 종료 시 검은 창(콘솔)을 닫으면 됩니다.

### DB 파일

- `noah_data.db`는 OneDrive를 통해 공유되며, 대시보드가 자동으로 인식합니다.
- 자동 인식이 안 되는 경우: `dashboard_config.ini` 파일을 열어 DB 경로를 직접 입력합니다.

```ini
[database]
db_path = C:\Users\홍길동\OneDrive - Rotork plc\...\noah_data.db
```

### 사이드바 필터

모든 페이지에 공통 필터가 적용됩니다:

- **시장 구분**: 전체 / 국내 / 해외
- **연도 / 월**: 기간 필터
- **섹터**: 복수 선택 가능
- **고객**: 복수 선택 가능
- **데이터 새로고침**: 캐시 초기화 (5분마다 자동 갱신)

## 3. 페이지별 안내

### 오늘의 현황

당일 기준 주요 Action Items:

- **KPI 카드**: 오늘 납기 / 이번 주 납기 / 금월 수주 / 금월 출고
- **납기 캘린더**: 월별 히트맵, 날짜 클릭 시 EXW 출고 예정 / 공장 픽업 / 납기 예정 / 출고 실적 상세
- **EXW 출고 지연**: EXW 일자가 지났으나 공장 출고 미완료 건
- **해외 선적 Action Items**: 포워더 미정 / 선적 대기 건
- **세금계산서 미발행**: 출고 완료 but 세금계산서 미발행 건 (Aging 차트 + 고객별 금액)

### 발주 커버리지

PO Status 기반 발주 현황:

| 상태 | 의미 |
|------|------|
| 미발주 | PO 없음 또는 Open (공장 발주 전) |
| 부분 발주 | 일부 Open + 일부 Sent/Confirmed |
| 발주 진행중 | Sent (공장에 발주, 확인 대기) |
| 발주 확정 | Confirmed / Invoiced |

- 출고 완료 건, PO 전체 Cancelled 건은 분석에서 자동 제외

### 수익성 분석

SO 매출 vs PO 원가(ICO) 마진:

- 원가 미확정 건(PO 없음)은 마진 계산에서 제외되며 별도 안내
- 저마진 경보: 마진율 20% 미만 건 Top 10

### Order Book (3탭)

- **Executive**: 워터폴 차트 (월별 선택 / 누적 토글), Backlog Cover, Past Due Ratio, Book-to-Bill
- **Risk**: Aging 분석, 고금액 위험건 Top 10 (납기 지연 중 금액 큰 순), 납기 히트맵
- **Conversion**: SO→PO→DN 전환 퍼널, 리드타임 분석 (KPI + Box plot + 동적 해석)

## 4. 향후 계획 — ERP 데이터 활용

현재는 NOAH 데이터만 대상이지만, 동일한 방식으로 ERP 데이터 활용이 가능합니다.

```
ERP (D365 / AX2009) → Excel/CSV 다운로드 → DB 동기화 → Dashboard
```

확장 가능 영역:

| 데이터 | 활용 |
|--------|------|
| Trial Balance (TB) | P&L, BS 월별 추이 대시보드 |
| AP/AR Aging | 미수금/미지급 현황 |
| 재고 데이터 | 재고 회전율, 과잉재고 분석 |

ERP에서 Excel/CSV로 데이터를 내려받으면, 원하는 형태의 분석 대시보드를 만들 수 있습니다.

---

## 5. 문제 해결

| 증상 | 해결 |
|------|------|
| bat 실행 시 바로 닫힘 | 콘솔 메시지 확인 — DB 경로 또는 파일 누락 |
| 빈 화면 (데이터 없음) | 데이터 새로고침 버튼 클릭 / DB 파일 경로 확인 |
| 포트 충돌 | 자동으로 빈 포트를 찾으므로 보통 문제 없음 |
| DB 자동 인식 안 됨 | `dashboard_config.ini`에 경로 직접 입력 |
